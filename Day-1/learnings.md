# Day 1: Exploring Agentic AI with NVIDIA Nemotron

## What I explored today

- Explored NVIDIA's AI model ecosystem.
- Learned about NVIDIA NIM APIs.
- Generated and understood the use of API keys.
- Experimented with the Nemotron Playground environment.
- Used the model's reasoning capabilities to learn Binary Search in C++.
- Explored multimodal capabilities such as text and image understanding.
- Understood the difference between traditional conversational AI and Agentic AI.

## Key Learnings

### Conversational AI
- Optimized for natural interactions.
- Great for content generation and answering queries.

### Agentic AI
- Designed for reasoning and planning.
- Can execute multi-step tasks.
- Can leverage tools more autonomously.
