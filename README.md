# 🚀 5 Days of Agentic AI

A hands-on journey of building AI applications using **Python**, **NVIDIA NIM**, **OpenAI SDK**, and **Prompt Engineering**.
This repository documents my learning - from making my first API call to building a modular **AI Study Mentor** with intelligent prompt routing and clean software architecture.

---

## Core(Objectives) 

* Learn AI application development
* Understand Prompt Engineering
* Build a modular AI Study Mentor
* Apply software engineering best practices
* Document the complete learning journey

---

## 🛠 Tech Stack

* Python
* NVIDIA NIM API
* OpenAI Python SDK
* Prompt Engineering
* Git & GitHub
* VS Code
* python-dotenv

## 📚 5-Day Journey

| Day   | Focus                   | Outcome                                  |
| ----- | ----------------------- | ---------------------------------------- |
| Day 1 | NVIDIA NIM Fundamentals | Learned Agentic AI and API basics        |
| Day 2 | API Integration         | Connected NVIDIA NIM using OpenAI SDK    |
